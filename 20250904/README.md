# NCHC &times; NCKU Digital Biology Bootcamp &mdash; 數位生物技術訓練營

## Event Focus / 活動重點

 -  生成式 AI 技術應用於蛋白質設計

    透過 NVIDIA BioNeMo 平台，實作 RFdiffusion 與 ProteinMPNN 等創新技術。

 -  基因定序資料分析與加速運算實作

    運用 Parabricks 工具，加速處理大規模基因資料，掌握分析流程最佳化。
